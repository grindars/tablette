# GridFu

https://github.com/evilmartians/slashadmin/issues/3

## Installation

Add this line to your application's Gemfile:

    gem 'grid_fu'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install grid_fu

## Usage

Do not forget to describe config options.
